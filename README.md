# Social Learning Review

A living systematic review of computational models of social learning,
with interactive brain maps, study networks, and a paper-by-paper modal.

## Live site

Once deployed to GitHub Pages, this will be available at:
https://princessamygdala.github.io/social-learning-review/

## What's here

- **`index.html`** — the main site. Self-contained: data inlined as base64,
  external libraries loaded from CDN. Open it directly in a browser, or
  serve via any static host.
- **`brain_per_region/`** — pre-rendered nilearn glass-brain views, one
  per canonical anatomical region (ACC, vmPFC, …) plus a study-weighted
  twin (`*_studies.html`) for each.
- **`brain_per_topic/`** — same idea but grouped by substantive research
  topic (Imitation, Trust, Mentalizing, …).
- **`walkthrough_v2_with_landscape.html`** — eleven-step animated
  explainer of how an individual learning agent becomes a
  social-learning agent.
- **`studies.json`** — companion data file (also inlined into
  `index.html` as base64).
